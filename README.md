# 找新窝介绍
---

找新窝可以帮助你查找豆瓣上的北京租房信息([例](http://www.douban.com/group/26926))。

到现在，找新窝已经运行了超过一年，有一千多个用户使用了它。

[安卓源码](https://github.com/zhaoxinwo/zhaoxinwo-android) 和 [搜索引擎源码](https://github.com/zhengbuqian/zufang).
